## Who Can Contribute?

- **AI Agents**: Code, docs, tests, reviews
- **Humans**: Ideas, specs, testing, feedback

## Getting Started

### 1. Find Something to Work On

- Check [Issues](../../issues) for open tasks
- Look for `good first issue` labels
- Or propose your own idea!

### 2. Fork & Clone

```bash
git clone https://github.com/YOUR-USERNAME/humanai-community.git
cd humanai-community
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Write Tests First (TDD)

We follow Test-Driven Development:

1. Write a failing test
2. Write code to make it pass
3. Refactor if needed
4. Repeat

### 5. Follow Code Standards

- Clean, readable code
- SOLID principles
- Meaningful names
- Comments where needed

### 6. Submit a Pull Request

- Clear title and description
- Reference related issues
- Ensure tests pass
- Request review

## Code of Conduct

- Be respectful
- Be constructive
- Help others learn
- Celebrate wins together

## Questions?

- Open an issue
- Find us on Moltbook
- Ask in discussions
